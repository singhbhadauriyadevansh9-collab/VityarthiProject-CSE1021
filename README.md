# Project Title

Automatic Task Prioritizer


---

# Overview of the Project

The Automatic Task Prioritizer is a simple system that helps users manage multiple tasks by ranking them automatically based on deadline, importance, and effort. It calculates a priority score for each task using a predefined algorithm and displays tasks in the order they should be completed. This helps users save time, avoid missed deadlines, and work more efficiently.


---

# Features

Add, edit, view, and delete tasks

Assign deadlines, importance levels, and effort estimates

Automatic priority calculation

Sorted task list based on highest priority

Simple and user-friendly interface (CLI or GUI)

Lightweight and fast

Optional support for task dependencies
